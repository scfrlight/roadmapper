# 🚀 Roadmapper

An AI-powered strategic planning application built with the "Emergent" agentic stack. Roadmapper transforms your visions into actionable, high-fidelity roadmaps.

## ✨ Features
- **AI Architect**: Generates multi-phase strategic roadmaps from natural language.
- **Agentic Researcher**: Automatically identifies resources and tools for each milestone.
- **Premium UI**: Sleek, glassmorphic design featuring dark mode and smooth animations.
- **Cross-Platform**: Built with React Native and Expo for Android and iOS.

## 🛠 Tech Stack
- **Frontend**: React Native, Expo Router, NativeWind (Tailwind CSS)
- **Backend**: Supabase (Auth, PostgreSQL)
- **State**: Zustand
- **Build**: Expo Application Services (EAS)

## 📦 Getting the App
This repository is configured with GitHub Actions to automatically generate an **Android APK** on every push to the `main` branch.

1. Go to the **Actions** tab in this repository.
2. Select the latest successful "Build Android APK" run.
3. Download the `roadmap-apk` artifact.

## 🚀 Development
To run this project locally:
```bash
npm install
npx expo start
```
